To run this FastAPI project, first install the required dependencies by running python3 -m pip install fastapi uvicorn, which installs FastAPI and Uvicorn. Then start the backend server using uvicorn api_app:app --reload, where api_app is the name of your Python file (api_app.py) and app is the FastAPI instance inside it. Once the server is running, it will be available at http://127.0.0.1:8000 and the automatic API documentation can be accessed at http://127.0.0.1:8000/docs. If the server does not start, make sure you are in the correct project directory, the file is named api_app.py, and the FastAPI instance is called app, and if needed you can try uvicorn main:app --reload but only if your file is actually named main.py.

To run this FastAPI project on any system (Mac or Windows), first install the required dependencies by running python -m pip install fastapi uvicorn (on some Windows setups you may need py -m pip install fastapi uvicorn instead). Once installed, start the backend server using uvicorn api_app:app --reload, where api_app is the name of your Python file (api_app.py) and app is the FastAPI instance inside it; if that command doesn’t work, use python -m uvicorn api_app:app --reload (or py -m uvicorn api_app:app --reload on Windows). After the server starts, it will run at http://127.0.0.1:8000 and the interactive API documentation will be available at http://127.0.0.1:8000/docs, and if anything fails make sure you are in the correct project folder, the file is named correctly, and the FastAPI instance is called app.
